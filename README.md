# shidoan-sakusei｜中学校数学 学習指導案サポート

中学校数学科の **学習指導案** を、選択中心の操作でスピーディーに作成できる Web アプリです。
初任の先生が手を止めがちな「単元観・生徒観・指導観」「評価規準の文末」「本時の展開」を、
学習指導要領（平成29年告示）に沿ったテンプレートから選ぶだけで骨格が整い、
Word（.doc）出力や、生成AIに清書させるためのプロンプト出力まで行えます。

> 「アプリで骨格を8割、生成AIで仕上げを2割」というワークフローを想定しています。

## 主な機能

- **5ステップで指導案を作成**：基本情報 → 単元について → 学習過程 → 評価規準 → プレビュー
- **中学校数学 全22単元**（1〜3年・4領域）の単元目標テンプレート
- **単元観・生徒観・指導観**：必要な項目だけチェックで開き、文例を選択（各15例）。プレビューでは箇条書きではなく文章としてつなげて出力
- **本時の展開ビルダー**：表示する項目（学習活動／学習内容／主な発問／予想される生徒の反応／指導上の留意点／教師の手立て／評価／学習形態）を選び、右の選択肢リストから **クリック追加＋ドラッグ＆ドロップで順序入れ替え・重複可**
- **「〇〇」自動置換**：目標・評価規準の「〇〇」を単元名（または任意の語句）に自動で差し込み
- **単元別バンク**：本時で扱う問題・例題／予想されるつまずき・誤答と手立て
- **出力**：
  - 🤖 **AI清書プロンプトをコピー**（ChatGPT 等にそのまま貼り付け）
  - 📄 **Word保存（.doc）**：A4縦・明朝・罫線付きの表組みを保持（マクロ不要）
  - 🖨 **印刷 / PDF**、📋 **テキストコピー**

## 使い方（ローカル実行）

```bash
npm install
npm run dev
```

表示された `http://localhost:5173` をブラウザで開きます。

### ビルド / プレビュー

```bash
npm run build      # dist/ に出力
npm run preview    # ビルド結果をローカル確認
```

## GitHub Pages で公開する（任意）

1. `npm run build` で `dist/` を生成
2. リポジトリの Settings → Pages で、`dist/` を配信する設定（GitHub Actions もしくは `gh-pages` ブランチ）を行う
3. `vite.config.js` の `base: "./"` によりサブパス配信に対応済み

## ディレクトリ構成

```
shidoan-sakusei/
├─ index.html                              # エントリ HTML
├─ main.jsx                                # React マウント
├─ LessonPlanApp.jsx                       # アプリ本体（単一コンポーネント）
├─ package.json
├─ vite.config.js
├─ サンプル指導案.doc                        # 出力サンプル
└─ 検証レポート_1年目教員の利用シナリオ.md     # 設計の検討メモ
```

## 技術

- React 18 + Vite
- 外部UIライブラリ不要（スタイルはインライン）。Word出力もライブラリ不要（HTML→.doc 方式）
